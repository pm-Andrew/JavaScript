**Note Taking app that can be used by any note taking style. **

## Tasks
This assignment will have you create a client-side JavaScript application that demonstrates the use of the Web Storage API.
You must demonstrate the ability to:
add items to local storage
retrieve items from local storage
delete items from local storage
You have complete creative freedom and should create something that is meaningful that you would be proud to show a future employer or a customer.
