## Option 1 - List of People
This option will have you create a JSON file that contains data about people and then display the data as HTML on a web page
Create a JSON file and name it option1.json
Create a property named members that stores an array of objects
The array must contain five elements that are objects that contain the following properties and values:
    firstName with the value of a person's first name
    lastName with the value of a person's last name
    URL with a link to an image of a person on the web (absolute URL)
    hobbies with an array that contains three hobbies the person has
    Create a property named miscellaneous that is an object that has the following properties and values:
    favoriteBand and set it to the name of a band
    sfavoriteFood and set it to the value of food such as pizza, Greek, Italian, etc.

## Code Your HTML/CSS and JavaScript
Now it's time to develop your code.
All of your JavaScript code must be placed in the script.js file in a folder named js.
All of your CSS must be placed in a folder named css.
