## Interactive Emoji Mood Board
Check it out, folks! Time to create a mood board that speaks in emojis. Click on an emoji, 
and watch it grow in size or pop up with a funky caption. Let those emojis do the talking, 
and bring some digital vibes to life on the screen!

## What does it do?
Well glad you asked, I provided three buttons with emojis correlating to an "emotion" or projected feeling, the choices I provided are (zombie 🧟‍♂️, "happy"😃, and pizza 🍕🍕).
