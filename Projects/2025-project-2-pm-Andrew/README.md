# Order Form with JS Validation

A professional order form with fully custom JavaScript validation — no reliance
on the HTML `required` attribute. Includes dual error buckets (above and below
the form), focus management on the first invalid field, and submit button
state control.

## Features
- Validates all fields: text inputs, select menus, radio buttons,
  checkboxes, and textarea
- Error buckets render above and below the form on failed submission
- Submit button disables on submit, re-enables after validation runs
- Focus moves to the first unfilled field automatically
- All 50 states in the state select
- Fully responsive layout
- Passes WAVE with no errors, alerts, or contrast errors
