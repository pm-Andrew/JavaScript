# Weather Dashboard

A two-mode weather app using the OpenWeather API. Displays current conditions
and a 5-day / 3-hour forecast for any location, with custom styling and
additional data points beyond the base requirements.

## Features

- Current weather: temperature, feels like, high/low, humidity, wind speed, plus 2 additional data points
- 5-day forecast with the same data set, triggered by a separate button
- Forecast displayed differently from the provided starter code
- All JavaScript commented throughout
- Passes WAVE with no errors, alerts, or contrast errors
- HTML and CSS validated
