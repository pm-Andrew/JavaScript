# Penguin Nation Dashboard

A dynamic content dashboard built with vanilla JavaScript and Bootstrap.
Displays a rotating quote board, a formatted news feed, and a featured
student spotlight — all driven by JavaScript array data and timed intervals.

## Features

- Quotes rotate every 15 seconds via `setInterval()`
- Featured student cycles every 10 seconds
- Clicking the logo clears all active intervals
- News items rendered dynamically from an array into the DOM
- Fully responsive layout using Bootstrap
- Passes WAVE and Accessibility Insights FastPass with no errors or alerts

## Technologies

- JavaScript (ES6)
- Bootstrap 5
- HTML5 / CSS3
